## Running

### Clone & install

* Clone this repo
```sh
git clone https://github.com/Negin-ozn/sample-note-taking.git
```
* `cd sample-note-taking`


Run
```sh
yarn install
```

### iOS

*  `cd ios`
  
Run
```sh
pod install
```
*  `cd ..`
  
Run
```sh
npx react-native run-ios
```

### Android

Run
```sh
npx react-native run-android
```
